### 4. Обучение и сравнение моделей

Обучила модели Linear Regression, Linear Regression со стандартизацией через скалер, Lasso, Ridge, ElasticNet
Для моделей с регуляризацией использовался GridSearchCV с 10 фолдами.

Используемые метрики:
MSE (Mean Squared Error) — средний квадрат отклонения или абсолютная ошибка
R² — относительное качество модели
Business-metric — доля предсказаний, отличающихся от истинного значения не более чем на 10% по модулю ( сравниваются результаты по целевой переменной)

### 5. Полученные результаты и интерпретация

Простая линейная регрессия уже показала адекватное качество (R² ~ 0.6) 
Стандартизация сама по себе не дала значимого прироста качества, но позволила корректно интерпретировать коэффициенты
Lasso не занулил коэффициенты при дефолтном alpha, то есть все признаки несут информацию нет коллиниарности и шумов
Ridge-регрессия с подбором alpha дала: наиболее стабильные результаты, лучший баланс между качеством и устойчивостью модели.
По бизнес-метрике именно Ridge-модель показала нхороший результат на тестовых данных 
cамые высокие показатели бизнесовые на линейной модели ( я дописала нум онли, потому что была чистка под втолько вещественные столбцы)
